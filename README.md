# egegen-project

Backend için adımlar:
1) "composer install"
2) ".env" dosyası database bağlantısı için değiştirilecek
3) "php artisan migrate"
4) "php artisan db:seed"
5) "php artisan serve"

Frontend için adımlar:
1) "npm install"
2) "ng serve"
3) Browserınızdan "http://localhost:4200" linke tıklayın
